# Project Rules

- JANGAN pernah menjalankan perintah build atau deploy seperti:
  - `npx next build`
  - `npm run build`
  - `yarn build`
  - `pnpm build`
  - `next build`
- JANGAN mengubah konfigurasi production tanpa persetujuan.
- Jika perlu menjalankan build, tanyakan dulu ke user.
- Fokus hanya pada development mode (`npx next dev`, `npm run dev`, dll).
